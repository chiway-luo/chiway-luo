## 📌 仓库介绍

[omnihand_mujoco_moveit2](https://github.com/chiway-luo/omnihand_mujoco_moveit2.git) 使用 MuJoCo 物理引擎和 MoveIt2 运动规划框架,开发了一个适用于 智元-Omnihand 灵巧手的仿真平台
